If funded, this project democratizes advanced experimental design, allowing
Bonsai users to translate natural language prompts into error-free, highly
optimized workflows in minutes. Offloading systems engineering to an autonomous
loop lets researchers scale assay complexity without manual bottlenecks,
hardware synchronization delays, or memory allocation glitches.

Contributions to the Bonsai Ecosystem and Bonsai.ML: The Bonsai MCP server will
drive adoption across the entire ecosystem (e.g., Open Ephys, Harp, UCLA
Miniscope, Neurophotometrics, BonVision, DeepLabCut), with an immediate,
transformative impact on real-time machine learning via Bonsai.ML. Most Bonsai
users lack the machine learning expertise required to seamlessly integrate
complex ML models into their rigs or optimally tune their parameters. The
Bonsai MCP server lowers this barrier by clearly demonstrating how Bonsai.ML
packages can elevate specific workflows and directly assisting users with
optimal model parameterization.

AI Enablement and Large-Scale Optimization: An AI agent embedded in the
experimental control loop can be leveraged to assume full parametric control
over long-duration experiments. By continuously monitoring the live state of
the experimental environment and cross-referencing data with online inferences
from its broad knowledge base, the system can discover optimal experimental
trajectories. While autonomous execution loops are rapidly transforming other
fields—such as materials science (A-Lab), chemical synthesis (RoboRXN), and
synthetic biology (Biofoundries)—this paradigm has not yet been applied to
neuroscience. This platform could demonstrate that AI-driven experimental
optimization is fundamentally superior to rigid, human-designed, rule-based
control.

Our long-term vision is the complete automation of hypothesis-driven scientific
discovery. By routing multi-modal experimental results back to the LLM, the
system can leverage a vast biomedical literature corpus to interpret data,
synthesize novel mechanistic hypotheses, and design and run the next
experimental iteration via a structured human-in-the-loop validation safeguard.
While achieving complete autonomous control or full closed-loop automation is
beyond our current scope, this project takes the critical first steps toward
making self-driving neuroscience experiments a reality.
